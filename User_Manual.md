# FieldSense User Manual

This manual explains how to use the FieldSense web application to monitor assets and manage maintenance.

## 1. Getting Started
1. Navigate to the application URL in your browser.
2. Log in using your credentials provided by the Administrator.

## 2. The Dashboard
*   **Live Page:** Displays a real-time card for every refrigerator. Green indicates normal temperature, and Red indicates a threshold breach.
*   **Gauges:** Each card shows the current temperature and the safe range.

## 3. Manager Workflow (Fault Management)
1. When a sensor detects an anomaly, a notification appears on the **Live** page.
2. Go to the **Assign Tech** page.
3. You will see a list of "Open" issues.
4. Select a technician from the dropdown and click "Assign".

## 4. Technician Workflow (Resolving Faults)
1. Log in to your technician account.
2. Go to your **Notes/Issues** page to see tasks assigned to you.
3. Update the ticket status as you work (e.g., "In Progress").
4. Once resolved, add your **Resolution Notes** and mark the ticket as "Closed".

## 5. System Indicators
*   **WS LIVE (Green):** You are receiving real-time updates.
*   **POLLING (Red):** The system is checking for updates every 10 seconds.
